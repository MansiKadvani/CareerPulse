▣ CareerPulse – AI Placement Analytics System 

- CareerPulse is an AI-powered career analytics dashboard that predicts  student placement chances, salary ranges, and identifies weak skill areas using real academic and skill data.

- It also generates a professional analytics dashboard using Python, Pandas, NumPy, and Matplotlib.

---------------------------------------------------------------------------

---------------------------------------------------------------------------

◎ Project Goal

- CareerPulse is an AI-based Career Intelligence System that analyzes student academic and technical efforts to predict:

    ▰ Placement probability
    ▰ Salary range
    ▰ Weak skill zones
    ▰ Skill improvement insights

---------------------------------------------------------------------------

---------------------------------------------------------------------------

▤ System Workflow

    01 -> Load dataset using Pandas	
    02 -> Clean dataset (missing values, duplicates, datatypes)	
    03 -> Skill Normalizer (0–1 scaling)	
    04 -> Placement Score Calculation	
    05 -> Salary Range Prediction	
    06 -> Weak Skill Detection	
    07 -> Visualization using Matplotlib	
    08 -> Final Dashboard Generation	

---------------------------------------------------------------------------

---------------------------------------------------------------------------

▦ Visualization Suite

    - Bar Chart    ->   Top 10 placement performers
    - Pie Chart    ->	Weak skill distribution
    - Line Chart   ->	Placement score trend
    - Histogram    ->	Placement score distribution
    - Scatter Plot ->	Placement score & internships vs salary
    - Subplot      ->   Dashboard	All charts in one unified view

---------------------------------------------------------------------------

---------------------------------------------------------------------------

∑ Intelligence Formulas

    - Placement Score (0–100) = (gpa * 25) + (internship * 15) + (certification * 10) + (projects * 10) + (soft_skills * 15) + (aptitude * 25)

    - Salary Prediction
        Score	  |  Salary Range
        85–100	  |  10–18 LPA
        70–84	  |  6–10 LPA
        50–69	  |  3–6 LPA
        Below 50  |  Less than 3 LPA
    
    - Weak Skill Rule
            Any parameter below 60% of its maximum value is considered WEAK.
